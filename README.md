# 🔐 Miniguia de Estudos em Cibertecnologia

## ✨ Contexto e Objetivos
Este repositório foi criado como parte do desafio da DIO para explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa.  
O tema escolhido para este caderno temático é **Cibertecnologia**, área que envolve segurança digital, proteção de dados e tecnologias aplicadas à defesa cibernética.  

**Objetivos de estudo:**
- Entender os fundamentos da cibertecnologia e segurança da informação.
- Explorar boas práticas de proteção de dados e privacidade.
- Documentar experimentos com prompts aplicados ao tema.
- Criar um miniguia reutilizável para revisões futuras e apoio em projetos.

## 🧠 Engenharia de Prompts e "Cicatrizes"
Durante o processo, foram elaborados e testados diferentes prompts estratégicos.  
Aqui estão alguns exemplos documentados:

- **Prompt inicial:** "Explique os principais tipos de ataques cibernéticos."
  - **Resposta obtida:** lista ampla, mas pouco detalhada.
  - **Dificuldade:** resposta superficial → ajuste necessário.

- **Prompt ajustado:** "Liste os 5 principais tipos de ataques cibernéticos e explique cada um em até 3 linhas."
  - **Resposta obtida:** explicações objetivas e úteis para revisão.
  - **Observação:** resultado mais prático para estudo.

- **Cicatriz aprendida:** Prompts vagos geram respostas genéricas. É essencial delimitar escopo e pedir exemplos práticos.

## 📖 Miniguia de Estudo (Entrega Final)

### 🔹 Resumos Estruturados
- **Phishing:** técnica de fraude que busca obter dados sensíveis por meio de mensagens falsas.
- **Ransomware:** malware que bloqueia acesso a sistemas e exige pagamento para liberação.
- **Firewall:** sistema que controla tráfego de rede e protege contra acessos não autorizados.
- **Criptografia:** processo de codificação de informações para garantir confidencialidade.
- **VPN:** rede privada virtual que protege a comunicação e aumenta a privacidade online.

### 📑 Glossário
- **Malware:** software malicioso que compromete sistemas.
- **Zero-Day:** vulnerabilidade desconhecida explorada por atacantes.
- **Pentest:** teste de invasão para avaliar segurança de sistemas.
- **SOC:** centro de operações de segurança responsável por monitorar ameaças.

### 💬 Prompts Reutilizáveis
- "Explique [conceito de cibertecnologia] em linguagem simples para iniciantes."
- "Crie um resumo em tópicos sobre boas práticas de segurança digital."
- "Liste os principais ataques cibernéticos e como preveni-los."
- "Compare criptografia simétrica e assimétrica em até 5 linhas."
